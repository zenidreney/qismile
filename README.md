# Qi Smile

Qi Smile is a front-end portfolio project of a digital wellness portal. This site is designed and developed to showcase modern front-end web technologies and responsive design best practices.

## Tech Stack

- **HTML5**: Semantic markup for a robust structure
- **CSS3**: Custom styles and theming
- **Bootstrap 5**: Responsive grid, components, and utilities ([MIT License](https://opensource.org/licenses/MIT))
- **Bootstrap Icons**: Scalable vector icons
- **JavaScript (ES6 modules)**: Dynamic content loading and UI interactions
- **Design Principles**: Mobile-first, accessibility-aware layout

## Portfolio Features

- **Landing Page**: Clean introduction section with social media badges and easy navigation.
- **Videos & Music**: Embedded multimedia using responsive iframes and Bootstrap cards.
- **Articles**: Informational blocks styled with Bootstrap utilities and custom classes.
- **Daily Inspiration**: Dynamic quotes and messages rendered via JavaScript module.
- **Reusable Components**: Navbar and footer auto-loaded as HTML modules via JavaScript.
- **Custom Favicon**

## Folder Structure Highlights

- `/index.html` — Main portfolio landing page
- `/styles.css` — Custom style overrides
- `/components/` — Modular HTML for Navbar & Footer
- `/scripts.js` — Loads components and handles page interactivity
- `/media/` — Site images and favicon

## Quick Start

Clone the repository and open `index.html` in your browser. No backend or build steps are required.
